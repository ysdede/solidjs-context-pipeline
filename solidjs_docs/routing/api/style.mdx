`Style` is a component that adds the [`style`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/style) element to your document's `head`.

```tsx twoslash
import { Style } from "@solidjs/meta";

;
```

## Usage

### Adding `style` tag

The `Style` component can add CSS to style elements within your application.
It is recommended to add styles in an external stylesheet and use a `Link` instead, rather than using this component, however.

:::tip[Note]
	Styles within the `Style` component are not scoped.
:::

```tsx twoslash
import { MetaProvider, Style } from "@solidjs/meta";

export default function Root() {
	return (
		
		</MetaProvider>
	);
}
```
